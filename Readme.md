# 🌍 Countries API Project (React)

A modern React application that displays information about countries using the REST Countries API.
The project includes search functionality, shimmer loading effect, and light/dark mode toggle for a better user experience.

This project demonstrates API integration, React state management, UI loading states, and theme switching.

## 🚀 Features

🌎 Display list of countries from API

🔎 Search countries by name

🏳️ Show country flags

📊 Display population, region, and capital

✨ Shimmer loading effect while fetching data

🌙 Light / Dark mode toggle

📱 Responsive UI

## 🛠️ Tech Stack

⚛️ React.js

JavaScript (ES6+)

CSS

REST Countries API

## ✨ Key Concepts Implemented

API data fetching in React

Conditional rendering

Shimmer loading UI

Dark / Light theme toggle

Component-based architecture
